# Instagram static feed assets

Tato složka slouží pro ručně spravovanou (statickou) galerii.

## Jak aktualizovat galerii
1. Nahrajte nové soubory jako `01.jpg` až `06.jpg`.
2. Upravte `instagram-feed.json` (src, alt, link) podle potřeby.
3. Commit + push.

Pokud obrázek chybí, UI automaticky ukáže placeholder s ikonou kamery.
